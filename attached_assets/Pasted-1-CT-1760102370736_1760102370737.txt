1️⃣ 현재 구조의 강점과 한계
✅ 강점

등급/포인트/쿠폰 요약 구조는 이미 리텐션 포인트를 잘 잡았음

“내 혜택 한눈에”, “내 단골 매장” 섹션 분리가 논리적임

하단 CTA 버튼(모든 단골보기 / 쿠폰함 바로가기) 명확

❌ 한계
구분	문제	UX 영향
시각적 몰입감 부족	정보는 있는데 감정적 ‘리워드 느낌’이 약함	“내가 단골이구나”라는 자부심 부족
매장 정보 단조로움	로고·사진 없이 텍스트 위주	매장 식별력↓, 정서적 거리↑
인터랙션 없음	클릭, 슬라이드, 확장 등 동작 없음	체류시간 짧음
혜택 요약 중복	포인트/쿠폰이 두 번 언급됨	구조적 중복감
개인화 결핍	“최근 방문한 매장”, “리뷰 미작성” 등은 단순 카운트	사용자 행동 기반 추천 부족
🧭 2️⃣ 리디자인 핵심 방향
목표	설계 방향
💎 감정적 몰입	등급/포인트 영역을 “명예의 뱃지화” (시각적 강화)
🏪 매장 중심	매장 썸네일/로고/카드 뷰 중심 구조로 전환
🔁 재방문 유도	“최근 주문한 매장”, “다시 가기” CTA 추가
🧠 개인화	사용자 데이터 기반 “가장 많이 방문한 매장”, “혜택 갱신된 매장” 강조
⚡ 즉시성	쿠폰/리뷰/포인트 내역을 실시간 반영형 카드로
🎨 디자인 톤	밝고 따뜻한 TL 시그니처 톤 (#FF8A00) + 미니멀 화이트
🧱 3️⃣ 새로운 페이지 구조 제안 (TL 단골매장 v2)
1. 헤더
❤️ 단골매장     (🔍 검색)   (🛎 알림)
“총 8곳 | 최근 방문 2일 전”


→ 아이콘은 작고 심플하게, 헤더는 고정 (스크롤시 축소)

2. 나의 단골 요약 카드 (Hero Section)

큰 카드 1장 형태로 변경👇

👑 준혁님은 현재 플래티넘 등급이에요!
💰 누적 포인트: 24,300P | 🎟 보유 쿠폰: 5장


배경: TL 오렌지 → 화이트 gradient

등급별로 카드 색상 변경 (Bronze–Silver–Gold–Platinum)

등급 진행도 Progress Bar 추가 (예: 다음 등급까지 2,000P 남음)

버튼: [포인트내역] [쿠폰함]

➡️ 감정적 보상 강화 + 시각적 집중력↑

3. 내 혜택 한눈에

지금의 3칸 구조는 유지하되 아이콘 + 카드 인터랙션 강화

카드	행동	애니메이션
포인트 내역	클릭 시 상세 페이지	Fade-in modal
보유 쿠폰	클릭 시 쿠폰함 이동	Slide-right
미작성 리뷰	클릭 시 리뷰작성 리스트	Bounce 표시

➡️ 사용자가 탭만 눌러도 바로 이동할 수 있게.

4. 최근 방문 / 자주 찾는 매장

“단골”의 감정을 불러일으키는 핵심 섹션

📍 최근 방문한 매장
┌────────────────────────┐
│ 🍜 본격 로스터리 카페 [GOLD] │
│ “3일 전 방문 · 포인트 3,200P” │
│ [다시 주문] [리뷰 남기기] │
└────────────────────────┘


매장 사진 or 로고 삽입 (썸네일 1:1 원형)

방문일, 포인트, 쿠폰 상태 표시

액션 버튼은 2개 이하 (주문 / 리뷰)

리스트는 3~5개까지만 노출 → “모두보기” 버튼으로 확장

5. 내 단골 매장 전체 보기

탭 구조 제안👇

[전체] [한식] [카페] [주점] [배달가능]


태그별 필터링

각 카드에 “단골 등급 / 포인트 / 쿠폰 수 / 방문일”

카드 클릭 시 renderStore(store_id) 실행

6. 하단 CTA

고정형 sticky 버튼

🧾 모든 단골 보기   🎟 내 쿠폰함 바로가기


버튼 사이에 경계선 or 얇은 그림자 추가
→ 고정이지만 이질감 없는 자연스러운 존재감 유지

💡 5️⃣ 추가 콘텐츠 제안
